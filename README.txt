Información: El programa utiliza la API de pokemon https://pokeapi.co/ para buscar información de un pokemon. 
El usuario ingresa el nombre de un pokemon, por ejemplo: pikachu, charmander, squirtle, etc.. y al hacer clic en el botón se realiza una consulta a la API 
y se muestra la información. El programa utiliza fetch para hacer una solicitud HTTP a la API y traer los datos del pokemon y por ultimo muestra la informacion 
en el programa.
